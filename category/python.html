Pelican + github 个人博客

安装 Pelican

Pelican is a static site generator, written in Python.

Pelicon's github: https://github.com/getpelican/pelican/

Pelican's doc: http://docs.getpelican.com/en/3.6.3/

Pelicao's demo: Pelican + Github 搭建自己的静态博客

使用 Pelican

新建项目

$ pelican-quickstart

命令运行后生成项目目录结构

编写 md 源文件

生成 html 文件

  1. 将编写的 md 源文件，放到 content 目录下
  2. 执行 "pelican content" 命令
  3. 生成的网站文件会在 output 目录下

发表 github user pages

  1. 将上步中生成的 output 目录下文件全部上传至 github 的 user pages 项目中即可
  2. 访问 github 的 user pages 地址访问网站，默认为 username.github.io
